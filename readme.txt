This plugin converts Azeri characters "ə,ğ,ö,ü,i,ı,ç,ş" in post slugs to Latin characters.

== Description ==

**This plugin is available for reference only. Its functionality has been replaced and extended by [Az-To-Lat](http://wordpress.org/extend/plugins/aztolat/).**

*The original author of this plugin is Anton Skorobogatov, who passed away prematurely in 2007. I decided to make it available for installation through the Plugin Browser/Installer. I take no credit for the creation of this plugin.*

Converts Azeri characters in post slugs to Latin characters. Very useful for Azeri-speaking users of WordPress.

Thanks to Aydin Cavadli for the idea.

== Installation ==

1. Upload `aztolat` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
